#Git spurningar og svör

1. Hvað gera eftirfarandi Linux skipanir?
	cd er change directory, þú notar það til þess að breyta hver þú ert.
	cd ~ færir mann í heimdrif sitt.
	ls stendur fyrir list og það birtir lista yfir möppum og skrám þar sem þú ert.
	pwd stendur fyrir print working directory, það birtir möppuna sem þú ert að vinna í.
	mkdir stendur fyrir make directory, það er notað til þess að búa til nýja möppu.

2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone /staðsetning/möppu er notað til þess að gera afrit af möppu.
	git log er notað til þess að fá skrá yfir breytingum
	git status birtir skrár sem þú hefur breytt og þær sem þú átt eftir að bæta við í
	git diff birtir muninn á þeim skrám sem þú ert að reyna að sameina

3. Hver er munurinn á eftirfarandi git skipunum:
	a) birtir allar sameiningar árekstur
	b) birtir allan árekstur á grunn skrá
	c) birtir breytingar áður en þú sameinar

4. 	Hvað er útgáfustýring (e.version control)? 
	útgáfustýring er kerfi sem skráir breytingar af skrám yfir tíma svo þú getur sótt tilbaka sérstakar útgáfur seinna.

5.	Hverjir eru helstu kostir við að nota GIT?
	Kostir GITs eru það að GIT virkar mjög hratt. Það er auðvelt að búa til geymslur. Geymslur eru geymdar í bara einni möppu, það er ekki bætt við .svn möppu í hverja einustu möppu sem er í kóðanum þínum. Það er auðvelt að "branch"-a. Er gert fyrir GitHub sem er bara yndislegt.

6.	Hversu oft telur þú að eigi að gera commit í verkefni, rökstuddu?
	Hversu oft það ætti að gera commit í verkefni fer eftir hvort það sé einhver annar að fara yfir það. Það er auðveldara að fara yfir verkefnið ef þú notar commit oft og á góðan hátt. Það lítur betur út. Það er kannski ekki jafn mikilvægt þegar þú ert að vinna í einhverju sem engin mun fara yfir eða mun aldrei vera notað.

7.	Hvað er "Working directory"?" "Staging area"?" og "Repository" í GIT?
	Working directory er staðsetningin sem þú ert að vinna í. Staging area er bara staður þar sem þú ræður hvaða breytingar verða að raunveruleika. Repository er bara geymsla fyrir öll gögnin þín.

8.	Hvenær er sniðugt að nota greinar (branches)? 
	Þegar þú villt prufa einhvað.
